# Requisitos #



| ID   | Requisito                       | Descrição                                                                       |
| ---- | ------------------------------- | ------------------------------------------------------------------------------- |
| RF01 | Cadastro de Materiais           | O sistema deve permitir cadastrar novos materiais no estoque.                   |
| RF02 | Listagem de Materiais           | O sistema deve permitir visualizar todos os materiais cadastrados.              |
| RF03 | Atualização de Materiais        | O sistema deve permitir editar as informações de um material cadastrado.        |
| RF04 | Remoção de Materiais            | O sistema deve permitir excluir materiais do sistema.                           |
| RF05 | Cadastro de Categorias          | O sistema deve permitir cadastrar categorias para organizar os materiais.       |
| RF06 | Movimentação de Estoque         | O sistema deve permitir registrar entradas e saídas de materiais do estoque.    |
| RF07 | Histórico de Movimentações      | O sistema deve permitir visualizar o histórico de movimentações de estoque.     |
| RF08 | Visualização de Inventário      | O sistema deve permitir visualizar o inventário completo de materiais e ativos. |


| ID    | Requisito            | Descrição                                                                                   |
| ----- | -------------------- | ------------------------------------------------------------------------------------------- |
| RNF01 | Interface Responsiva | A interface deve ser responsiva e funcionar em diferentes tamanhos de tela.                 |
| RNF02 | API REST             | O sistema deve possuir uma API REST desenvolvida com Spring Boot.                           |
| RNF03 | Banco de Dados       | O sistema deve utilizar um banco de dados relacional para armazenamento das informações.    |
| RNF04 | Desempenho           | O sistema deve responder às requisições em tempo adequado para uso escolar.                 |
| RNF05 | Segurança            | O sistema deve proteger os dados contra acessos não autorizados.                            |
| RNF06 | Usabilidade          | A interface deve ser simples e intuitiva para facilitar o uso pelos funcionários da escola. |
